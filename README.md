# carelink-ux-case-study

CareLink is a concept UX/UI case study focused on solving real-world healthcare scheduling and appointment management challenges.

This project explores:
- Patient-side appointment booking experience
- Admin & staff scheduling workflows
- Reducing no-shows and scheduling conflicts
- Designing scalable healthcare systems
